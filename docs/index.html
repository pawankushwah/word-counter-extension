Word Counter Extension

Get the Chrome extension that counts words in real-time as you select text on any webpage.

Download Extension (.crx)

Important Security Notice:

Chrome shows warnings for extensions downloaded outside the Chrome Web Store. This is normal for directly hosted extensions. Follow the steps below carefully to install.

Installation Instructions

Since this extension is hosted directly on GitHub and not the Chrome Web Store, you'll need to follow a few manual steps to install it in your Chrome browser.

  • Download the `.crx` file: Click the "Download Extension (.crx)" button above. Chrome will download the file to your "Downloads" folder and might show a warning. Do NOT click "Discard" if prompted; choose "Keep" or similar.
  • Open Chrome Extensions Page: In your Chrome browser, type chrome://extensions into the address bar and press Enter.
  • Enable Developer Mode: On the Chrome Extensions page, locate the "Developer mode" toggle switch in the top-right corner of the page and turn it ON.
  • Drag and Drop: Find the downloaded `word_counter_extension.crx` file. You can usually see it in the downloads bar at the bottom of your Chrome window. Drag this `.crx` file from your downloads bar (or from your computer's "Downloads" folder) and drop it anywhere onto the `chrome://extensions` page.
  • Confirm Installation: A confirmation dialog will appear, asking you to confirm the installation. Click "Add extension" to complete the process.
  • Pin the Extension (Optional): For easy access, click the puzzle piece icon (Extensions icon) in your Chrome toolbar, find "Selected Word Counter," and click the pin icon next to it.

For Future Updates

This extension is configured to check for updates periodically. When a new version is released, Chrome will automatically update it in the background.
